🎬 Netflix-Style Movie Streaming App (Nuxt + Vue 3 + TMDB API)
I built a fully-functional Netflix-style streaming interface powered by real movie data from TMDB.
 The project focuses on speed, clean UI/UX, state management, and a smooth browsing experience similar to modern streaming platforms.
🚀 Key Features
Dynamic Home Page Sections
 Trending Movies • Top Rated • Popular • Weekly Highlights
 All sections load real data from TMDB in real-time.
Movie Details Page
 Each movie includes:
High-quality poster & backdrop
Rating & release info
Overview
Image preview
Watch Now interaction
Favorites System (Fully Persistent)
 Powered by Pinia + pinia-plugin-persistedstate
Add/remove movies to Favorites
Favorites saved automatically in Local Storage
Consistent UI state across all pages
Heart button syncs instantly across the app
Reusable FavoriteButton Component
Smart state detection
Animated toggle
Works in all movie sections (Trending, Top Rated, Details Page, etc.)
Responsive Modern UI
 Built with Tailwind CSS
 Fully optimized for mobile, tablet, and desktop.
🔧 Tech Stack
Nuxt 3 + Vue 3 (Composition API)
Pinia State Management
Tailwind CSS
Axios
TMDB API
FontAwesome Icons
